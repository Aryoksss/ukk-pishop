- Live previews
- Fullscreen mode
- Cross platform


## Documentation

[Documentation](https://laravel.com/docs/12.x)


## Installation

Install pishop

```bash
  git clone https://github.com/Aryoksss/ukk-pishop
  cd pishop
```
    
## Running

To run tests, run the following command

```
  npm run dev
```
```
  php artisan serve
```


## Tech Stack

**Client:** React, Laravel, TailwindCSS

**Server:** Nginx
